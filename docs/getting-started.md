# Getting Started with MultiClaw Public Library

## Who this is for
- builders joining MultiClaw lanes
- technical collaborators evaluating architecture
- operators who need quick, accurate runbooks

## 10-minute orientation
1. Read `departments/overview.md`
2. Review `stack/tooling-index.md`
3. Open the playbook relevant to your role
4. Cross-reference the implementation repo

## Writing standards
- be concrete and verifiable
- prefer links over duplicated specs
- separate current-state facts from future plans
- include assumptions and constraints
